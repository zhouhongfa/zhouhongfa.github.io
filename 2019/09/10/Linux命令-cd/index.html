Linux命令-cd

发表于 2019-09-10 更新于 2019-10-11 分类于 Linux命令

Linux cd 命令可以说是Linux中最基本的命令语句，其他的命令语句要进行操作，都是建立在使用 cd 命令上的。

所以，学习Linux 常用命令，首先就要学好 cd 命令的使用方法技巧。

命令格式

cd [目录名]

命令功能

切换当前目录至dirName
